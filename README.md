## Usage

### Install dependencies
```
cd frontend
npm install
cd ..   
```
```
npm install
```

### Run Both React Dev Server and Backend Server using

```
npm run dev
```

• Developed a user friendly todo list with create, update, delete and mark as complete functionalities.<br>
